# Bilibili 影片下載 + 內容轉 Markdown 系統

## 📋 需求分析

| 項目 | 說明 |
|------|------|
| **來源** | Bilibili 播放清單 (space.bilibili.com/1925268550/lists/839662) |
| **功能** | 1. 批量下載全部影片<br>2. 提取音訊轉文字<br>3. 生成 Markdown 檔案 |
| **輸出** | 每條片一個 .md 檔案，包含轉錄內容 |

## 🛠️ 技術方案

### 方案 A: Python + yt-dlp + Whisper (推薦)

| 組件 | 工具 | 用途 |
|------|------|------|
| 影片下載 | yt-dlp | 支援 Bilibili，可下載播放清單 |
| 音訊提取 | FFmpeg | 從影片提取音軌 |
| 語音轉文字 | OpenAI Whisper | 本地運行，支援中文 |
| Markdown 生成 | Python script | 格式化輸出 |

**優點**:
- ✅ 完全開源免費
- ✅ 支援批量處理
- ✅ 可自訂輸出格式
- ✅ 本地運行，私隱安全

**缺點**:
- ❌ 需要安裝 Python 環境
- ❌ Whisper 較大模型需要 GPU/較長時間

---

### 方案 B: 現成工具組合

1. **下載**: you-get / annie / JDownloader
2. **轉文字**: 剪映專業版 (免費語音識別)
3. **整理**: 手動或簡單腳本

**優點**:
- ✅ 圖形界面，易用
- ✅ 剪映中文識別準確

**缺點**:
- ❌ 難以自動化批量處理
- ❌ 剪映需要人手操作

## 🚀 建議方案：Python 自動化腳本

### 系統要求

| 項目 | 最低要求 | 建議配置 |
|------|----------|----------|
| OS | Windows 10 / macOS / Linux | Windows 11 / Ubuntu |
| Python | 3.9+ | 3.11+ |
| RAM | 8GB | 16GB+ |
| 儲存 | 10GB 可用空間 | 50GB+ (視乎影片數量) |
| GPU | 可選 | NVIDIA GPU (加速 Whisper) |

### 需要安裝嘅軟件

```bash
# 1. Python 依賴
pip install yt-dlp openai-whisper ffmpeg-python

# 2. FFmpeg (系統級)
# Windows: choco install ffmpeg
# macOS: brew install ffmpeg
# Ubuntu: sudo apt install ffmpeg

# 3. Whisper 模型 (首次運行自動下載)
# tiny: ~39 MB (快但準確度一般)
# base: ~74 MB
# small: ~244 MB (建議)
# medium: ~769 MB
# large: ~1550 MB (最準但慢)
```

## ⏱️ 預計處理時間

| 步驟 | 1條 10分鐘片 | 50條片 (總計) |
|------|-------------|--------------|
| 下載 | 2-5 分鐘 | 2-4 小時 |
| 提取音訊 | 30 秒 | 25 分鐘 |
| Whisper tiny | 1 分鐘 | 50 分鐘 |
| Whisper small | 3 分鐘 | 2.5 小時 |
| Whisper medium | 8 分鐘 | 6-7 小時 |
| 生成 Markdown | 5 秒 | 4 分鐘 |
| **總計 (small)** | **~7 分鐘** | **~6-8 小時** |

> 備註：時間取決網絡速度、影片長度、電腦性能
