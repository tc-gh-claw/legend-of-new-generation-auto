# 共融橋樑 | Inclusion Bridge

## SDG 10 - 減少不平等 (Reduced Inequalities)

一款幫助用戶透過日常行動減少社會不平等的應用程式。

---

## 設計理念

### 視覺參考
- **Irma Boom** 書籍設計的層次感與節奏
- **Karel Martens** 實驗性印刷的溫暖質感
- **人權設計運動** 的包容性設計原則

### 色彩系統
- **赤陶色 (#E07A5F)** - 溫暖、親切、行動力
- **沙色 (#F2E9DC)** - 柔和、包容的背景
- **炭灰色 (#3D405B)** - 沉穩的文字色
- **金色 (#F4A261)** - 希望與成就
- **鼠尾草綠 (#81B29A)** - 完成狀態

## 功能特色

### 🎯 共融指數儀表板
- 圓環進度視覺化
- 累積行動統計
- 連續天數追蹤

### 🤝 每日共融行動
四個維度的日常行動：

1. **平等教育** 📚
   - 了解弱勢群體的故事或歷史

2. **社區連結** 🤝
   - 與不同背景的人進行真誠對話

3. **支持本地** 💪
   - 購買或分享弱勢群體製作的產品

4. **發聲支持** 📢
   - 在社交媒體分享平等相關資訊

### 💡 共融錦囊輪播
每日輪播啟發性語錄，激發用戶持續行動

### 📝 共融日誌
記錄個人的共融行動與反思

## SDG 10 背景

**目標：** 減少國家內部和國家之間的不平等

不平等不僅關乎經濟，更涉及機會、權利和尊嚴。透過日常的小行動，每個人都能為建立更公平的社會作出貢獻。
