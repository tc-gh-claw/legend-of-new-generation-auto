### v1.9.0 (2026-03-22 10:30 PM)
**新增功能：冰霜元素敵人 (Ice Elemental Enemy)**

#### 修改文件

   - `js/scenes/BattleScene.js` - 新增冰霜元素戰鬥機制和冰凍狀態

   - `index.html` - 更新版本號至 v1.9.0

#### 功能詳情

**1. 敵人基礎屬性**
- **敵人名稱**：冰霜元素 (Ice Elemental)
- **圖標**：❄️
- **等級要求**：玩家等級 ≥ 21 時才會出現
- **生成位置**：迷霧森林 (ForestScene)

**2. 基礎數據**
```javascript
'enemy-ice-elemental': {
    name: '冰霜元素',
    hp: 120,
    maxHp: 120,
    attack: 22,
    exp: 95,
    gold: 65,
    isIceElemental: true,
    freezeChance: 0.4,
    iceBurstChance: 0.35,
    freezeSkipChance: 0.35
}
```

**3. 與火焰元素的對比**

| 屬性 | 火焰元素 | 冰霜元素 |
|------|----------|----------|
| HP | 90 | 120 |
| 攻擊力 | 30 | 22 |
| 類型 | 攻擊型 | 防禦型 |
| 特殊效果 | 燃燒（持續傷害） | 冰凍（跳過回合） |
| 經驗值 | 85 | 95 |
| 金幣 | 55 | 65 |

**4. 特殊技能**

**❄️ 技能一：冰凍攻擊 (Freeze Attack)**
- **觸發機率**：40%
- **效果**：
  - 造成 10-16 點冰霜傷害
  - 附加「冰凍」狀態，持續3回合
  - 每回合有 35% 機率無法行動
- **視覺效果**：
  - 冰晶環繞玩家
  - 「❄️ 冰凍者」標籤顯示
  - 冰凍指示器在畫面左上角顯示

**❄️ 技能二：極寒冰爆 (Ice Burst)**
- **觸發機率**：35%（當冰凍狀態未激活時）
- **效果**：
  - 造成 18-25 點冰霜傷害
  - 必定附加冰凍狀態
  - 每回合有 50% 機率無法行動（更高機率）
- **視覺效果**：
  - 全屏藍色閃光
  - 20個冰晶粒子四散
  - 中央大型冰爆特效
  - 玩家受擊閃爍（藍色）

**5. 冰凍狀態機制**
```javascript
freezeEffect: {
    active: false,
    duration: 0,
    skipTurnChance: 0
}
```
- **持續時間**：3回合
- **跳過回合機率**：
  - 普通冰凍：35%
  - 極寒冰爆：50%
- **觸發時機**：玩家回合開始時
- **狀態指示器**：畫面左上角顯示冰凍圖標
- **消退提示**：冰凍結束時顯示「❄️ 冰凍效果消退了！」

**6. 冰凍與燃燒的區別**

| 效果 | 燃燒 | 冰凍 |
|------|------|------|
| 傷害類型 | 固定每回合傷害 | 無直接傷害 |
| 影響方式 | 每回合扣HP | 有機率跳過回合 |
| 策略性 | 需要治療 | 需要快速擊殺 |
| 視覺 | 紅色火焰 | 藍色冰晶 |

**7. 森林場景行為**
- **外觀特徵**：
  - 青藍色調 (0x00ffff)
  - 緩慢脈動動畫（比火焰慢）
  - 周圍環繞6個旋轉冰晶
- **巡邏模式**：
  - 較慢的移動速度（比火焰慢）
  - 持續釋放冰晶粒子效果
  - 偵測範圍：120像素（比火焰短）
  - 追逐速度：45（比火焰慢）

**8. 戰鬥特性**
- **高HP (120)**：冰霜元素是防禦型敵人
- **中等攻擊力 (22)**：比火焰元素低
- **冰凍標記**：戰鬥界面顯示「❄️ 冰凍者」標記
- **特殊背景**：戰鬥背景為冰藍色調，帶有雪花粒子
- **額外獎勵**：擊敗後獲得額外 30 經驗值和 25 金幣

**9. 視覺效果**

**戰鬥外觀：**
- 青藍色調染色
- 「❄️ 冰凍者」標籤顯示在名稱下方
- 周圍環繞6個反向旋轉冰晶
- 緩慢閃爍動畫
- 冰霜光環效果

**冰凍特效：**
- 10個冰晶環繞玩家旋轉
- 「凍結！」文字提示
- 玩家變藍動畫
- 冰凍指示器閃爍效果

**極寒冰爆特效：**
- 中心大冰爆（青色光圈擴散）
- 20個冰晶粒子向四周飛散
- 全屏藍色閃光
- 「❄️ 轟隆！」文字提示

**10. 戰鬥背景效果**
- 冰藍色漸變背景
- 30個雪花粒子緩慢飄落
- 冰霜覆蓋效果（脈動透明度）
- 與火焰元素的紅色背景形成對比

- **高HP (120)**：冰霜元素是防禦型敵人，能承受更多傷害
- **較低攻擊力 (22)**：比火焰元素低，但控制效果強
- **冰凍狀態**：跳過回合機制增加戰鬥不確定性
- **高獎勵 (95exp, 65gold)**：匹配其高難度和高HP
- **等級限制 (Lv21+)**：確保玩家有足夠實力應對
- **技能冷卻**：冰凍狀態不重疊，給玩家喘息機會

- **優先擊殺**：冰霜元素的控制效果很危險，應優先處理
- **快速解決**：利用高連擊快速擊殺，避免被冰凍拖延
- **時間凍結**：使用時間凍結技能跳過危險回合
- **風險評估**：冰凍的跳過回合是機率性的，有時可以賭一把
