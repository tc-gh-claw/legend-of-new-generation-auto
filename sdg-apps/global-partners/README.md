# 全球伙伴 Global Partners

一款促進 SDG 17「促進目標實現的伙伴關係」的應用程式。

## 設計理念

**參考設計師：**
- Kenya Hara (無印良品) - 極簡與留白
- Susan Kare (早期 Mac 圖標) - 友好直觀的圖標系統
- 聯合國視覺識別系統 - 專業與信任感

**色彩方案：**
- 主色：SDG 17 深藍色 (#19486a)
- 強調：金色 (#fcc30b) 象徵連結與價值
- 背景：乾淨的白色與淺灰

## 功能

1. **伙伴關係指數儀表板** - 可視化你的連結網絡
2. **四大伙伴行動** - 分享、連結、學習、支持
3. **伙伴日誌** - 記錄你的合作歷程
4. **每日伙伴錦囊** - 激發連結靈感
5. **網絡視覺化** - 個人→社區→組織→全球的連結層次

## SDG 17 簡介

> 加強執行手段，重振全球永續發展伙伴關係

這是最後一個 SDG，也是實現其他 16 個目標的關鍵。沒有全球合作，就無法實現永續發展。
