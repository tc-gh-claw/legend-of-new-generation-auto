# 🎵 音效與音樂資源指南

本目錄用於存放遊戲的音效和背景音樂文件。

│   ├── combat/             # 戰鬥音效
│   │   ├── attack.wav      # 攻擊音效
│   │   ├── hit.wav         # 受擊音效

│   │   ├── victory.wav     # 勝利音效
│   │   ├── defeat.wav      # 失敗音效
│   │   └── levelup.wav     # 升級音效
│   ├── magic/              # 魔法音效
│   │   ├── math.wav        # 數學技能
│   │   ├── science.wav     # 科學技能
│   │   ├── english.wav     # 英文技能
│   │   ├── general.wav     # 常識技能
│   │   ├── heal.wav        # 治療技能
│   │   └── shield.wav      # 防禦技能
│   └── environment/        # 環境音效
│       ├── footstep.wav    # 腳步聲
│       ├── encounter.wav   # 遭遇敵人
│       ├── save.wav        # 存檔音效

    │   ├── menu.ogg        # 主選單音樂

## 📝 使用建議

### 音效格式
- **音效**: 建議使用 `.wav` 格式（無損，快速載入）
- **音樂**: 建議使用 `.ogg` 格式（壓縮比高，音質好）
- 亦可使用 `.mp3`（兼容性最好）

### 音量建議
- 背景音樂: 0.6-0.8 (60-80%)
- 音效: 0.7-1.0 (70-100%)
- UI音效: 0.5-0.7 (較輕柔)

### 文件命名規範
- 使用小寫字母
- 用連字符分隔單詞
- 前綴區分類型：`sfx-`、`bgm-`

## ⚠️ 授權注意事項

使用任何第三方資源前，請務必：

1. **檢查授權條款** - 確認允許商業使用
2. **確認署名要求** - 部分資源需要標註作者
3. **保留授權文件** - 在項目中記錄使用的外部資源
4. **優先選擇 CC0** - 公共領域資源最靈活

常見授權類型：
- **CC0**: 公共領域，可做任何用途
- **CC-BY**: 需要署名原作者
- **CC-BY-SA**: 需要署名且衍生作品使用相同授權
- **CC-BY-NC**: 非商業使用
