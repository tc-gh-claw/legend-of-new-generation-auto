# 社區互助橋 (Community Bridge) 🤝

**對應 SDG：** SDG 1 - 無貧窮 (No Poverty)

## 設計理念

參考設計風格：
- **IDEO.org** 人本設計原則 - 以用戶為中心的設計，注重可及性與包容性
- **Paula Scher** 的視覺識別設計 - 大膽但溫暖的配色傳達希望與力量
- **社會企業界面** - 簡潔、無障礙、易於使用

配色方案：
- 主色：溫暖紅色 (#E5243B) - 代表關懷與行動
- 輔色：大地棕色 (#8B4513) - 代表穩定與信任
- 強調色：暖金色 (#D4A574) - 代表希望與價值
- 背景：溫暖米色 (#FDF8F3) - 創造舒適、無壓力的氛圍

## 核心功能

### 💰 簡易預算追蹤
- 清晰顯示本月預算與剩餘金額
- 視覺化進度條（安全/警告/危險狀態）
- 快速記錄收入與支出

### 🏛️ 社區資源導航
- 食物銀行位置與開放時間
- 免費技能培訓課程
- 社區健康服務
- 職業介紹及配對

### 🎯 脫貧目標追蹤
- 個人化目標設定
- 進度可視化
- 完成獎勵系統

### 🏅 成就徽章系統
- 起步者徽章
- 記帳達人徽章
- 儲蓄新手徽章
- 技能提升徽章
- 互助之星徽章
- 脫貧勇士徽章

### 💡 每日理財錦囊
輪播式理財提示，包括：
- 「先儲蓄，後消費」原則
- 支出記錄的重要性
- 善用社區資源

## SDG 對齊

### 目標 1.1
消除各地一切形式的貧窮
- 提供社區資源資訊，連結有需要人士與援助

### 目標 1.2
降低貧窮人口比例
- 透過預算管理與目標追蹤，協助用戶改善財務狀況

### 目標 1.4
確保所有人獲得基本服務
- 彙整社區支援服務資訊，提高可及性

### 目標 1.B
建立適當的社會保障制度
- 鼓勵儲蓄習慣，建立個人財務安全網

## 技術實現

- 純 HTML5 + CSS3 + Vanilla JavaScript
- 響應式設計，優先支援手機使用
- 無需後端，可離線使用
- 本地儲存（可擴展 LocalStorage）
