# 新世代傳說

> **Legend of the New Generation** - 全科知識RPG冒險遊戲

[![GitHub Pages](https://img.shields.io/badge/Live%20Demo-GitHub%20Pages-blue)](https://tc-gh-claw.github.io/legend-of-new-generation/)
[![Phaser 3](https://img.shields.io/badge/Engine-Phaser%203-orange)](https://phaser.io/)
[![License](https://img.shields.io/badge/License-MIT-green)]()

## 🎮 遊戲概念

結合小學全科知識（數學、科學、英文、常識）的像素風回合制RPG遊戲。玩家在魔法世界中冒險，透過回答問題來施放技能、打敗敵人、拯救世界！

## ✨ 核心特色

- 🎨 **像素風美術** - 復古8-bit/16-bit風格
- ⚔️ **回合制戰鬥** - 經典RPG戰鬥系統
- 📚 **全科整合** - 數學、科學、英文、常識
- 🏆 **角色成長** - 升級、裝備、技能樹
- 🗺️ **開放世界** - 多個區域探索
- 💾 **進度保存** - LocalStorage存檔

## 🎯 學科職業系統

| 學科 | 職業 | 特色 |
|------|------|------|
| 🔢 數學 | **邏輯法師** | 精準計算型攻擊 |
| ⚗️ 科學 | **元素煉金術士** | 火/水/風/土元素魔法 |
| 📖 英文 | **語言吟遊詩人** | 輔助、治療、增益 |
| 🌍 常識 | **知識守衛者** | 防禦型、知識護盾 |

## 🚀 開始遊戲

### 在線遊玩
👉 [https://tc-gh-claw.github.io/legend-of-new-generation/](https://tc-gh-claw.github.io/legend-of-new-generation/)

### 本地運行
```bash
git clone https://github.com/tc-gh-claw/legend-of-new-generation.git
cd legend-of-new-generation
# 使用任何本地伺服器，例如：
python3 -m http.server 8000
# 然後瀏覽 http://localhost:8000
```

## 🛠️ 技術棧

- **引擎**: [Phaser 3](https://phaser.io/) - HTML5遊戲框架
- **語言**: JavaScript (ES6+)
- **地圖編輯**: [Tiled](https://www.mapeditor.org/)
- **素材**: [OpenGameArt](https://opengameart.org/)
- **部署**: GitHub Pages

## 📁 項目結構

```
legend-of-new-generation/
├── index.html              # 主頁面
├── assets/                 # 遊戲素材
│   ├── images/            # 圖片素材
│   ├── audio/             # 音效音樂
│   └── tilemaps/          # Tiled地圖檔
├── js/                     # JavaScript代碼
│   ├── scenes/            # Phaser場景
│   ├── entities/          # 遊戲實體
│   ├── systems/           # 遊戲系統
│   └── data/              # 題目數據
├── css/                    # 樣式表
└── docs/                   # 文檔
```
