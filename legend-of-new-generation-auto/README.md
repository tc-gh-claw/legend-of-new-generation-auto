## 🤖 自動開發模式

這是「新世代傳說」的**實驗性自動開發版本**，每 2 小時由 AI 自動添加新功能：

## 🎮 遊戲概念

結合小學全科知識（數學、科學、英文、常識）的插畫風回合制RPG遊戲。玩家在魔法世界中冒險，透過回答問題來施放技能、打敗敵人、拯救世界！

## 🛠️ 技術棧

- **引擎**: [Phaser 3](https://phaser.io/) - HTML5遊戲框架
- **語言**: JavaScript (ES6+)
- **部署**: GitHub Pages + GitHub Actions 自動部署
- **自動化**: AI Cron Job 每2小時更新

## 📁 項目結構

