## 🆕 新增功能

### 暴擊藥水 (Critical Hit Potion)

**概念**: 新增一種消耗品道具，使用後可在下一場戰鬥中獲得額外的暴擊率加成，與現有的連擊獎勵系統形成完美搭配。

#### 道具詳情

| 屬性 | 數值 |
|------|------|
| 名稱 | 暴擊藥水 |
| 圖標 | 🎯 |
| 價格 | 250G (購買) / 125G (出售) |
| 效果 | 下場戰鬥暴擊率+30% |
| 類型 | 消耗品 |

#### 系統特性

1. **購買方式**: 可在雜貨店購買
2. **使用時機**: 戰鬥外使用，效果持續到下一場戰鬥結束
3. **效果疊加**: 可與連擊獎勵的暴擊加成疊加
4. **視覺反饋**: 
   - 使用時顯示紅色粒子特效
   - 背包中顯示「暴擊加成啟用中」狀態指示器
   - 戰鬥開始時顯示暴擊加成狀態提示
5. **自動消耗**: 戰鬥結束後（無論勝敗）自動清除效果

#### 暴擊率計算公式

```
最終暴擊率 = 基礎暴擊率(10% + 等級*2%) + 連擊獎勵加成 + 暴擊藥水加成
最高上限: 95%
```

**示例**: 
- 等級5玩家基礎暴擊率: 20%
- 使用暴擊藥水: +30%
- 達到5連擊: +15%
- **最終暴擊率**: 65%
