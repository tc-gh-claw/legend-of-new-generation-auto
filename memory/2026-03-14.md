## 💡 設計決策記錄

**選擇 Phaser 3 原因：**
- 最流行嘅HTML5遊戲引擎
- 大量教程同社區支持
- 免費開源
- 完美支持 GitHub Pages

**核心玩法循環：**
```
探索世界 → 遭遇敵人 → 選擇學科 → 回答問題 → 施放技能 → 獲得經驗 → 升級變強
```

**教育價值：**
- 將學習變成冒險
- 即時反饋（答對=攻擊成功）
- 多學科整合
- 進度成就感
