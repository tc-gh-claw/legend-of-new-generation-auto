# 🔧 創新工坊 | Innovation Workshop

## SDG 9: 產業、創新和基礎設施

一款鼓勵動手修理、DIY 創作和永續創新的應用。幫助用戶追蹤維修項目、記錄創意點子、管理工具材料，並計算通過「修復而非丟棄」所帶來的環境影響。從個人層面支持負責任的工業實踐和創新精神。

## 功能特性

### 🎯 核心功能

1. **維修項目追蹤**
   - 記錄待修物品（電器、家具、衣物等）
   - 項目狀態管理：待修→進行中→已完成
   - 維修難度評級和預估時間

2. **創新點子日誌**
   - 快速記錄靈感點子
   - 為每個想法添加標籤和分類
   - 追蹤點子從概念到實現的過程

3. **工具材料庫**
   - 管理現有工具和材料庫存
   - 借出/歸還記錄
   - 購買清單和 Wishlist

4. **環境影響儀表板**
   - 已挽救物品數量統計
   - 減少垃圾量估算
   - 節省金額計算
   - 累積成就感展示

5. **技能學習追蹤**
   - 記錄新學會的維修技能
   - 技能等級提升系統
   - 相關教學資源連結

6. **項目時光軸**
   - 回顧所有完成項目
   - 照片記錄維修前後對比
   - 分享維修故事

## 使用方式

直接在瀏覽器打開 `index.html` 即可使用。

### 快速開始
1. 添加第一個維修項目（如「修理漏水水龍頭」）
2. 記錄所需的工具和材料
3. 開始維修，更新項目狀態
4. 完成後記錄成果，查看環境影響統計
5. 累積技能徽章，成為維修達人！
